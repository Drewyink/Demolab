# EcholinkSolutions Facets-Style Sandbox (Training)

This is a training simulator inspired by Facets-style configuration workflows. It is NOT TriZetto/Facets and contains no proprietary code.

## Run locally
```bash
npm install
npm run initdb
npm start
```
Open: http://localhost:3000

## Default accounts
- Admin: admin@echolink.local / Passw0rd!
- Student: student@echolink.local / Passw0rd!
