💡 What Is a Claim Pend?
A pend occurs when a claim cannot auto-adjudicate because something is missing or flagged for review. A pend is NOT a denial. Fix the issue and the claim releases for payment. Most common reason: missing prior authorization.

Pend Resolution Guide
Pend ReasonResolution Steps
Missing PAAuthorizations module. Create auth for member, provider, and CPT code. Return to Claims. Set PA On File = Yes. Resubmit.
Provider Not FoundProviders module. Verify NPI is active and In-Network. Provider Contracts. Confirm active contract exists. Fix and resubmit.
Eligibility IssueMembers module. Confirm member is Active. Check coverage effective and term dates against service date. Fix and resubmit.
Duplicate ClaimCompare to original paid claim. If confirmed duplicate, deny with duplicate reason code. If different, override edit and release.
COB NeededMembers module. Add or verify other insurance information. Return to claim and reprocess with Non-Duplication COB applied.
