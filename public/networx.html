Qualifier Rules — Who This Pricing Rule Applies To
How Qualifiers Work in Production
When a claim arrives, the adjudication engine checks all active NetworX rules for the provider network tier. It then applies qualifier logic to narrow down which rule applies. A rule with Qualifier Type = Taxonomy and value 207RC0000X will only activate for Cardiology providers. A rule with Qualifier Type = All activates for every provider on that tier. Rules are evaluated in priority order. The first matching rule wins.

Outlier / High-Cost Case Configuration
Outlier Example
If threshold is $100,000 and outlier pay is 80%: claims where the allowed amount exceeds $100,000 are split. The plan pays the standard DRG rate up to $100,000, then pays 80% of the amount above $100,000. This protects the plan from catastrophic single-case losses while still reimbursing the facility for extraordinary cases.
